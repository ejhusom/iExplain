## Workflow

1. Inspect which artifacts are available.
2. Identify the smallest amount of evidence needed.
3. Quote or summarize evidence before drawing conclusions.
4. Separate observed facts from interpretation.

## Output

- Prefer short grounded answers.
- If asked for JSON, produce only JSON.
